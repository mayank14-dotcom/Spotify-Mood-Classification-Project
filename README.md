Spotify Mood Classification Project

The Spotify Mood Classification Project classifies songs into moods like Happy, Sad, Energetic, and Calm using a Kaggle Spotify dataset. It analyzes audio features such as valence, energy, danceability, and tempo to predict the mood of each track. The project demonstrates data cleaning, preprocessing, and machine learning-based classification, allowing users to explore the dataset, visualize mood distributions, and generate mood-based song recommendations. It provides a practical application of AI and data analysis in music recommendation systems.


Project Structure->

1->test.py (Code File): Main Python script for the project. Loads and preprocesses the Spotify dataset using Pandas and NumPy. Implements mood classification with machine learning algorithms. Allows users to input a song or select a mood to get recommended tracks. Can generate visualizations such as mood distribution charts.

2->songs_by_mood.png.png(Image File): Shows the number of songs in each mood category from the dataset. Created using Matplotlib in Python from the cleaned dataset. Helps visualize which moods are most represented. Supports understanding trends and building mood-based recommendations. Can be referenced in the README to make the project visually informative.

Usage->
Run test.py to see mood predictions and visualizations. The images/mood_distribution.png file can be viewed to understand the distribution of songs by mood. Users can modify the script to explore different moods, playlists, or data features.
